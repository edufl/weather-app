# 🌦️ Weather App

Um aplicativo de clima desenvolvido com React, que exibe o clima atual e a previsão de 7 dias para qualquer cidade.

---

## 🚀 Funcionalidades

🔎 Digite o nome de uma cidade e veja:

- 🌡️ Temperatura atual  
- 🌬️ Velocidade do vento  
- 🧭 Direção do vento  
- 📅 Previsão de 7 dias

## 🛠️ Tecnologias utilizadas

- ⚛️ React (useState)
- 🌐 Fetch API
- 🌦️ Open-Meteo API (sem necessidade de chave)
- 🎨 CSS puro

---

## 📡 APIs utilizadas

- Geocoding:
  https://geocoding-api.open-meteo.com/

- Weather:
  https://api.open-meteo.com/

## ⚙️ Como rodar o projeto

```bash
# Clonar o repositório
git clone https://github.com/edufl/weather-app.git

# Entrar na pasta
cd weather-app

# Instalar dependências
npm install

# Rodar o projeto
npm start
```
